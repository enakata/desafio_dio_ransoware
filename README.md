# desafio_dio_ransoware
## Conclusão e Aprendizados
### Reflexões sobre o Projeto:
* Entendimento Técnico: A implementação prática permitiu compreender profundamente como malwares funcionam em nível técnico.

* Importância da Prevenção: Desenvolver as contramedidas destacou como a prevenção é mais eficiente que a reação.

* Complexidade da Detecção: Mesmo com ferramentas simples, é possível implementar sistemas de detecção razoavelmente eficazes.

* Ética e Responsabilidade: O projeto reforçou a importância do uso ético desses conhecimentos.

### Recomendações de Segurança:

* Mantenha sistemas atualizados

* Eduque usuários sobre ameaças

* Implemente políticas de backup rigorosas
